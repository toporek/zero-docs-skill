# Zero 1.6

## Installation

```bash
npm install @rocicorp/zero@1.6
```

## Upgrading

### PlanetScale Failover

Previous Zero versions lost replication slots after a PlanetScale Postgres failover, forcing resync. Zero 1.6 fixes this problem. To enable support, see [High Availability and Failover](../connecting-to-postgres.md#high-availability).

## Features

* [**Litestream Region:**](../zero-cache-config.md#litestream-region) Added support for deployments in non-standard AWS partitions like GovCloud (thanks [@ericykim](https://github.com/ericykim)!).
