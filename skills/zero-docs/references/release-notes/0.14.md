# Zero 0.14

[Zero 0.14 release notes](https://zero.rocicorp.dev/video/releases/call-me-0.14-v1.mp4)

## Install

```bash
npm install @rocicorp/zero@0.14
```

## Features

* Use `from()` to map column or tables to a different name ([docs](../schema.md#name-mapping)).
* Sync from multiple Postgres schemas ([docs](../schema.md#multiple-schemas))

* `useQuery` not working when `server` unset ([bug](https://bugs.rocicorp.dev/issue/3497))

* `Row` helper doesn't work with query having `one()` ([bug](https://bugs.rocicorp.dev/issue/3503))
