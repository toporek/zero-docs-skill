# Quickstart

Minimal starter apps for Zero with a variety of stacks.

If you want the guided tutorial instead, see [Tutorial](tutorial.md).

All of the starters below have a similar UI and structure to demo Zero's capabilities, but they are built with different stacks and have different features:

[Zero-cache syncing between Postgres and SQLite](https://zero.rocicorp.dev/video/hello-zero-v1.mp4)

## hello-zero-solid

Simple starter for Zero and SolidJS.

**Stack:** Vite/Hono/SolidJS **Source:** [https://github.com/rocicorp/hello-zero-solid](https://github.com/rocicorp/hello-zero-solid)**Features:** Instant reads and writes, realtime updates

## hello-zero-cf

This starter runs Zero in a React/Hono app within the Cloudflare worker environment. It also runs `zero-client` within a Durable Object and monitors changes to a Zero query.

**Stack:** pnpm/Vite/Hono/React/Cloudflare Workers **Source:** [https://github.com/rocicorp/hello-zero-cf](https://github.com/rocicorp/hello-zero-cf)

## hello-zero

Simple starter for Zero and React.

**Stack:** Vite/Hono/React **Source:** [https://github.com/rocicorp/hello-zero](https://github.com/rocicorp/hello-zero)**Features:** Instant reads and writes, realtime updates.
