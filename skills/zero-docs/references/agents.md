# Agent Support

Zero's documentation is available as plain text for LLMs and AI coding assistants.

## Options

- **[/llms.txt](/llms.txt)**: Index of all documentation pages with descriptions and links.
- **[/llms-full.txt](/llms-full.txt)**: Complete documentation in a single file.
- **`/docs/{slug}.md`**: Any individual page as markdown (e.g., [/docs/introduction.md](/docs/introduction.md), [/docs/schema.md](/docs/schema.md)).
